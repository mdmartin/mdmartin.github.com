Rank the Team

This project is still being developed. I am a soccer fan and it is always a topic between fans that which team is better. It is not enough to only see the league table for this season. Through my machine learning course, I learnt some clustering algorithms, by which I can rank those teams with real data. I'm still developing the program and look for better clustering algorithm. Besides, I'm also seeking for an effective way to visualize the result.

Analysis of GRE Reading Passages

It is also not finished yet. This project is like the course project of Statistical Analysis of Language. Sometimes, that course was not that fun because of introducing statistical formulas which I had already learnt from another class. However, I think it is very attractive to extract useful information, analyze, and exploit them. I was shocked once by a group of MIT students who used their program to generate papers automatically. By my own little project, I want to analyze all GRE reading passages that I can find. The first step is to extract key words and sentence structures from them, which will benefit students who prepare for GRE. Second, by further study, I want to generate "GRE-like" passages automatically.
